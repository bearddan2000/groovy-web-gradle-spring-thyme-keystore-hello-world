# groovy-web-gradle-spring-thyme-keystore-hello-world

## Description
A springboot web app with thymeleaf support and self-signed ssl.

## Tech stack
- groovy
- gradle
  - springboot
  - thymeleaf
  - bootstrap
  - jquery
  - datatable

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`
[Available at](https://localhost)

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
